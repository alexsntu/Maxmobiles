# Команда: Блог

Назначение: запуск переработки статьи блога Maxmobiles через связку правила и скилла.

## Что использовать

- Skill: `seo-blog-article-builder-maxmobiles`
- Rule: `seo--for-blog-article-maxmobiles.mdc`
- CSS-база: `maxmobiles-styles.css`

## Инструкция для агента

1. Сначала прочитай и примени:
   - Rule: `.cursor/rules/seo--for-blog-article-maxmobiles.mdc`
   - Skill: `.cursor/skills/seo-blog-article-builder-maxmobiles/SKILL.md`
   - Reference: `.cursor/skills/seo-blog-article-builder-maxmobiles/ARTICLE-STANDARD.md`
   - CSS-базу: `maxmobiles-styles.css`
2. Прими исходный HTML статьи.
3. Запроси у пользователя 4 параметра:
   - slug
   - дата публикации (`YYYY-MM-DD`)
   - категория (`Лайфхаки и инструкции`, `Статьи`, `Новости`)
   - перелинковка (`Анкор -> URL`) или `нет`
4. Переработай статью строго по правилу `seo--for-blog-article-maxmobiles.mdc`.
5. Используй только классы `mm-*`, которые есть в `maxmobiles-styles.css` (без новых/самодельных классов).
6. Проверь изображения по стандарту:
   - первая картинка: `fetchpriority="high"` и без `loading="lazy"`
   - остальные картинки: `loading="lazy"`
   - у всех картинок: `<figure>`, `alt`, `title`, `style="max-width: 100%; height: auto; border-radius: 24px; display: block;"`
7. Сохрани результат в файл `Блог/<slug>.html`.
8. В чат выведи только краткое подтверждение и путь к файлу.
