# Команда: Сервис описание категории

Назначение: генерация информационного блока **Block 3** (этапы ремонта, гарантия, запчасти, стоимость, отзывы) для страниц ремонта Maxmobiles.

## Что использовать

- Skill: `service-info-block-builder`
- Rule: `service-info-block.mdc`
- Эталон: `Новые категории ремонта/iphone-15-pro-max-block-3.html`

## Инструкция для агента

1. Прочитай и соблюдай:
   - `.cursor/skills/service-info-block-builder/SKILL.md`
   - `.cursor/rules/service-info-block.mdc`
2. Бери за основу эталон `iphone-15-pro-max-block-3.html`: CSS и секцию отзывов копируй без изменений (4 карточки, ссылка на Яндекс Карты).
3. Если входных данных нет — запроси по таблице скилла: заголовок H2 этапов и **4 шага**, заголовок + текст гарантии, заголовок + текст запчастей, заголовок + текст стоимости.
4. Тексты пользователя вставляй **как есть**, без правок и перефразирования; несколько абзацев — отдельные `<p>`; списки со знаком `-` в плитке стоимости оформляй через `<ul class="mmb3-list"><li>...</li></ul>`.
5. Не меняй структуру HTML, классы `mmb3-*`, SVG-иконки, бейджи в плитках, порядок секций из правила.
6. Сохрани файл как `SLUG-block-3.html` в `Новые категории ремонта/` (SLUG по правилу скилла).
7. В чат выведи краткое подтверждение и путь к файлу.
