# Команда: Теги

Назначение: полный SEO-анализ текущих мета-тегов страницы Maxmobiles с конкурентной проверкой органики и рекомендацией финальных `H1`, `title`, `description`.

## Что использовать

- Skill: `seo-meta-analyzer`
- Rule: `seo--meta-tags.mdc`

## Инструкция для агента

1. Сначала прочитай и примени:
   - `.cursor/skills/seo-meta-analyzer/SKILL.md`
   - `.cursor/rules/seo--meta-tags.mdc`
2. Прими входные данные от пользователя:
   - страница/сущность (категория или товар)
   - текущие `H1`, `title`, `description` (или отметка, что тегов нет)
3. Проведи аудит текущих тегов по стандарту `seo--meta-tags.mdc`.
4. Выполни SERP-анализ только по органике (без маркетплейсов и агрегаторов), собери конкурентов и смежные LSI-запросы.
5. Учти гео-приоритет: Севастополь и Крым; доставку по России использовать как вторичный коммерческий сигнал.
6. Сформируй финальные рекомендации по `H1`, `title`, `description` так, чтобы покрыть максимум релевантного спроса категории/товара в Яндексе и Google, включая GEO/AEO-сигналы.
7. Формат финального блока тегов — строго по `seo--meta-tags.mdc`; отклонения допустимы только при обосновании данными выдачи.
