# Service Price Table Builder (Block 2)

Генерирует готовый CS-Cart HTML-блок с таблицей цен для страниц ремонта Maxmobiles.  
Кнопки открывают модаль из Block 1 через `mmb1OpenModal()`.

---

## Обязательные входные данные

Попроси если чего-то нет:

| # | Что | Пример |
|---|-----|--------|
| 1 | **Заголовок H2** | `Цена на Ремонт iPhone 15 Pro` |
| 2 | **Подзаголовок** | `У нас нет "хитрых" наценок...` |
| 3 | **Услуги** — одна строка на услугу (формат ниже) | — |
| 4 | **Гарантия по каждой услуге** | см. ниже |

После того как пользователь прислал список услуг — **обязательно спроси гарантию**, прежде чем генерировать HTML:

> Укажите гарантийный срок для каждой услуги (или напишите «нет» если гарантии нет):
> 1. Замена стекла — ?
> 2. Замена дисплея (копия) — ?
> …

---

## Формат строки услуги

```
Название – Время - URL (вид запчасти «Опция») [цена «от»] [гарантия «N дней»] - Описание
```

**Разбор элементов:**

| Элемент | Как определить |
|---------|----------------|
| **Название** | текст до ` – ` |
| **Время** | текст между ` – ` и первым ` - ` |
| **URL** | полный URL до `(` или до ` - Описание` если нет опции |
| **Опция** | содержимое `«»` внутри `(вид запчасти «...»)`, пустая строка если скобок нет |
| **Цена «от»** | `from: true` если в строке есть `цену пишем «от»` или `цена «от»` или в URL содержится `cena-ot` |
| **Гарантия** | содержимое `«»` внутри `(гарантия «...»)`, пустая строка если скобок нет — тогда бейдж не добавляется |
| **Описание** | текст после последнего ` - ` |

**Примеры разбора:**

```
Замена стекла – 30 мин - https://maxmobiles.ru/displey-zamena-stekla/ (гарантия «30 дней») - Описание...
→ warranty: "30 дней"  →  <span class="mmb2-svc-warranty">Гарантия 30 дней</span>

Замена дисплея (копия) – 30 мин - https://maxmobiles.ru/displey-original/ (вид запчасти «Копия») (гарантия «90 дней») - Описание...
→ warranty: "90 дней"

Чистка после воды – 30 мин - https://maxmobiles.ru/profilaktika-posle-vody/ - Описание...
→ warranty: ""  →  бейдж не добавляется
```

## Процесс генерации

Читай эталон: `Новые категории ремонта\iphone-15-pro-max-block-2.html`

### Шаги

1. **CSS и оформление** — копируй полностью без изменений (все `.mmb2-*` стили, liquid glass, dark borders).

2. **HTML шапки** — подставь H2 и подзаголовок от пользователя.

3. **Получи цены** — для каждой уникальной URL загрузи страницу через WebFetch. Из текста страницы извлеки цену:
   - Если `option: ''` — первая цена в теле страницы (рядом с заголовком товара)
   - Если `option: 'Копия'` / `'Аналог'` / `'Оригинал'` — найди строку `- Копия (NN ₽)` и возьми число из неё
   - Несколько услуг с одним URL — загрузи страницу один раз, извлеки разные опции
   
   Форматируй число с разрядами через `&nbsp;`: `22 000` → `22&nbsp;000`.

4. **Строки таблицы** — для каждой услуги генерируй `<tr>` по шаблону из [row-template.md](row-template.md). Нумерация ячеек цены: `mmb2-p0`, `mmb2-p1` … по порядку.
   - Если у услуги есть гарантия — добавляй **после** `.mmb2-svc-time`:
     ```html
     <span class="mmb2-svc-warranty">Гарантия 90 дней</span>
     ```
     Если гарантии нет — не добавляй ничего.
   - **Обычная цена:** `<span class="mmb2-price" id="mmb2-pN">22&nbsp;000&nbsp;₽</span>`
   - **Цена «от»** (`from: true`):
     ```html
     <span class="mmb2-price" id="mmb2-pN"><span style="font-size:13px;font-weight:500;color:#ff9500">от&nbsp;</span>9&nbsp;000&nbsp;₽</span>
     ```
     И в ячейке цены добавь: `<div class="mmb2-price-from-label">зависит от запчасти</div>`
   - `onclick` кнопки: `mmb2OpenModal('НАЗВАНИЕ', 'mmb2-pN')`

5. **JS-блок** — только функция `mmb2OpenModal`, без XHR/fetch. Копируй из эталона (раздел «JAVASCRIPT — открытие модали»). Никакого `ROWS`, `mmb2Fetch`, `mmb2ParsePrice`, `mmb2LoadPrices`.

6. **Имя файла** — `SLUG-block-2.html`, где SLUG = транслит устройства через дефис.  
   Сохрани в `Новые категории ремонта\`.

## Обновление цен в существующем файле

Когда пользователь говорит **«пересобери цены»** / **«обнови цены»**:

1. Открой файл `*-block-2.html`
2. Найди все URL в `href` первой колонки каждой строки таблицы
3. Загрузи каждый уникальный URL через WebFetch (параллельно)
4. Извлеки актуальные цены (с учётом опций, как описано в шаге 3 выше)
5. Перезапиши только содержимое `<span class="mmb2-price" id="mmb2-pN">…</span>` — ничего больше не трогай

## Визуальные стандарты таблицы

### Desktop

- **Первая колонка** — всегда `300px` (`colgroup col:nth-child(1)`)
- **Зебра-полосы** — `tbody tr:nth-child(even)` фон `#f7f7f9`
- **Hover** — `#eef4fd`
- **Ячейка цены** — `text-align: right` + `vertical-align: middle` (правый край создаёт вертикальную линию из ₽)
- **Сброс темы CS-Cart** — сразу после `border-collapse` добавляй:
  ```css
  .mmb2 .mmb2-table th,
  .mmb2 .mmb2-table td { border: none; outline: none; box-shadow: none; }
  ```

### Размеры шрифтов

| Элемент | Desktop | Mobile |
|---------|---------|--------|
| `.mmb2-svc-name` | `18px` | `16px` |
| `.mmb2-price` | `21px` | `20px` |
| `.mmb2-svc-time` | `12px` | `11px` |
| `.mmb2-svc-warranty` | `11px` | `10.5px` |

### Mobile (≤780px) — карточки + аккордеон

- `.mmb2-table-wrap` становится прозрачной (без фона/тени/скруглений)
- Каждая `<tr>` — белая карточка: `border-radius: 14px`, `box-shadow`, `padding: 13px 14px`
- Сетка строки: `grid-template-columns: 1fr 155px` (≤480px: `1fr 148px`) — фиксированная колонка цены выравнивает все строки
- Зебра и hover переопределены: все карточки белые
- Ячейки: `background: transparent; border: none; box-shadow: none`
- `.mmb2-price-from-label` на мобиле: `white-space: nowrap; max-width: none; text-align: right`

**Аккордеон:**
- `.mmb2-expand-btn` — `display: none` вне медиазапроса (скрыта на PC)
- На мобиле: `display: flex`, занимает `grid-column: 1/-1; grid-row: 2`, текст «Подробнее» / «Скрыть», шеврон через `::after`
- JS (функция `mmb2InitAccordion`) инжектирует кнопку в `<tr>` и переключает класс `.mmb2-open`
- Описание (`td:nth-child(3)`, row 3) и кнопка заявки (`td:nth-child(4)`, row 4) скрыты по умолчанию, показываются при `.mmb2-open`

## Что НЕ менять

- Всю CSS секцию (`.mmb2-table-outer`, `.mmb2-table-wrap`, liquid glass, dark borders, адаптив, аккордеон)
- Функции `mmb2OpenModal` и `mmb2InitAccordion` — копируй из эталона без изменений
- Структуру HTML: `mmb2-table-outer` → `mmb2-table-wrap` → `mmb2-table`
- Атрибут `onclick="mmb2OpenModal(...)"` на кнопках (не `data-repair-modal`)
- Описания, время, тексты строк таблицы при обновлении цен
